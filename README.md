## Bu uygulama ne işe yarar?

Mimari ofislerin arşivlerinde **on binlerce dosyayı** (proje çizimleri, render'lar, fotoğraflar, BIM modelleri, dokümanlar) **akıllı şekilde** bulup yönetmesini sağlar — tamamen **offline**, internet bağımlılığı yok.

| Özellik | Açıklama |
|---|---|
| **Dosya Tarama** | DWG, MAX, IFC, RVT, SKP, PDF + 95 format, otomatik kopya bulma |
| **AI Arama** | "Cumhuriyet dönemi cumbalı bina" gibi yazıp eşleşen görselleri/dosyaları bulun |
| **AI Sohbet** | Arşivinize soru sorun; dosya içeriğinden cevap üretir |
| **Önizleme** | DWG, 3D MAX, PSD, PDF, video — küçük resim otomatik oluşur |
| **Çoklu Arşiv** | Ana arşiv + yerel arşiv + `.archivistpro` export/import ile paylaşım |
| **Tamamen Offline** | İnternet sadece kurulum sırasında gerekir; dosyalar bilgisayardan çıkmaz |
| **5 Dil** | Türkçe · English · العربية · 日本語 · 中文 |

## Sistem Gereksinimleri

- **İşletim Sistemi:** Windows 10 / 11 (64-bit)
- **RAM:** 4 GB minimum (8 GB+ önerilen)
- **Disk:** 2 GB boş alan (5 GB+ önerilen)
- **Ekran:** 1366×768+ (1920×1080+ önerilen)

> **AI Özellikleri için:** [Ollama](https://ollama.com/download) + chat modeli (~5 GB) + vision modeli (~5 GB) opsiyoneldir. Olmadan da dosya tarama, arama ve kopya bulma çalışır.
