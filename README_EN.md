## What does this app do?

Helps architecture offices intelligently find and manage **tens of thousands of files** in their archives (project drawings, renders, photos, BIM models, documents) — fully **offline**, no internet dependency.

| Feature | Description |
|---|---|
| **File Scanning** | DWG, MAX, IFC, RVT, SKP, PDF + 95 formats, automatic duplicate detection |
| **AI Search** | Type "Republican-era bay-windowed building" and find matching images/files |
| **AI Chat** | Ask questions about your archive; answers generated from file content |
| **Preview** | DWG, 3D MAX, PSD, PDF, video — thumbnails auto-generated |
| **Multi-Archive** | Main + local archives + `.archivistpro` export/import for sharing |
| **Fully Offline** | Internet needed only during install; files never leave the computer |
| **5 Languages** | English · Türkçe · العربية · 日本語 · 中文 |

## System Requirements

- **OS:** Windows 10 / 11 (64-bit)
- **RAM:** 4 GB minimum (8 GB+ recommended)
- **Disk:** 2 GB free space (5 GB+ recommended)
- **Display:** 1366×768+ (1920×1080+ recommended)

> **For AI features:** [Ollama](https://ollama.com/download) + chat model (~5 GB) + vision model (~5 GB) are optional. File scanning, search, and duplicate-finding work without them.
