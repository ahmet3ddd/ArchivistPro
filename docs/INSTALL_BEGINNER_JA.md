# ArchivistPro — インストールガイド (初心者向け)

> **バージョン:** 3.0.0 | **日付:** 2026-05-23 | **プラットフォーム:** Windows 10/11 (64-bit)
>
> このガイドは、初めてコンピューターにプログラムをインストールする方や、インストール経験が少ない方向けです。各ステップはスクリーンショット付きで説明されています。専門用語はなく、事前知識も不要です。
>
> より簡潔な/技術的な要約は **[プロフェッショナルインストールガイド](https://github.com/ahmet3ddd/ArchivistPro/releases/download/v3.0.0/INSTALL_PRO_JA.md)** を参照してください。

---

## 1. 準備

### 1.1. お使いのコンピューターは準備できていますか?

ArchivistPro をインストールする前に、コンピューターが以下の要件を満たしているか確認してください。最近のほとんどの PC は余裕で超えています。

| 要件 | 最小 | 推奨 |
|---|---|---|
| オペレーティングシステム | Windows 10 (64-bit) | Windows 11 (64-bit) |
| RAM (メモリ) | 4 GB | 8 GB 以上 |
| 空きディスク容量 | 2 GB | 5 GB 以上 |
| プロセッサー (CPU) | x64 アーキテクチャ (ほとんどの Intel / AMD) | 4 コア以上 |
| ディスプレイ | 1366×768 ピクセル | 1920×1080 以上 |

> **コンピューターのスペックを確認するには?**
>
> 1. `Windows キー + R` を同時に押します
> 2. 小さなダイアログで `msinfo32` と入力し **OK** をクリックします
> 3. 「システムの概要」の下に OS、プロセッサー、RAM が表示されます

### 1.2. 管理者権限

プログラムをインストールするには、Windows に**管理者**としてログインしている必要があります。会社のコンピューターを使用していて「管理者権限が必要」と表示される場合は、IT 部門にお問い合わせください。

### 1.3. インターネット接続

インターネットは**インストーラーをダウンロードする時のみ**必要です。インストール後、ArchivistPro は**オフラインで動作します** — ファイルがコンピューターから外部に送信されることはありません。

## 2. インストーラーのダウンロード

![GitHub Releases ページ](img/install/github-releases.png)

1. Web ブラウザ (Chrome、Edge、Firefox) を開きます。
2. 次のアドレスに移動: **https://github.com/ahmet3ddd/Arsiv-H2/releases/latest**
3. 「**Assets**」セクションの下にファイルが表示されます。次の**いずれか**をダウンロードしてください:
   - **`ArchivistPro_3.0.0_x64_en-US.msi`** ← 推奨
   - `ArchivistPro_3.0.0_x64-setup.exe` ← 代替

> **MSI と EXE の違い:** どちらも同じプログラムをインストールします。MSI はより一般的で、会社のコンピューターでの管理が容易です。どちらでも問題ありません。

> **ブラウザが警告を出した場合:** 「このファイルはめったにダウンロードされません。本当ですか?」のようなメッセージが表示される場合があります。**保存**または**許可**をクリックして続行してください。

## 3. インストーラーの実行

![インストールウィザード](img/install/installer-wizard.png)

1. ダウンロードしたファイル (`ArchivistPro_3.0.0_x64_en-US.msi`) を**ダブルクリック**します。通常**ダウンロード**フォルダーにあります。
2. Windows が尋ねる場合があります: 「*このアプリがデバイスに変更を加えることを許可しますか?*」 → **はい**を選択します。
3. インストーラーウィンドウで:
   - **次へ**をクリック
   - ライセンスを読んで同意し、**次へ**
   - インストール場所をデフォルト (`C:\Program Files\ArchivistPro\`) のままにして**次へ**
   - **インストール**をクリック
4. 数秒待ちます。「セットアップが完了しました」が表示されたら**完了**をクリックします。

> **SmartScreen 警告が表示された場合:** 「Windows が PC を保護しました」というタイトルの青い画面が表示されることがあります。これはアプリケーションがコード署名プロセス中であるためです。次の手順を実行してください:
>
> 1. 「**詳細情報**」をクリックします
> 2. 「**実行**」をクリックします

インストール後、デスクトップに **ArchivistPro** ショートカットが表示されます。スタートメニューからも起動できます。

## 4. 初回起動 — セットアップウィザード

![セットアップウィザードステップ 1](img/install/wizard-step-1.png)

プログラムを初めて開くと、**5 ステップのセットアップウィザード**が起動します。一度しか表示されません。約 3〜5 分。

### ステップ 1 — 言語とシステムチェック

- インターフェース言語を選択: **日本語** (後で**設定**からいつでも変更できます)。
- プログラムがハードウェアを自動的にチェックします。エラーがなければ**次へ**をクリックします。

### ステップ 2 — ハードウェア検出

![ハードウェア検出](img/install/wizard-step-2.png)

- プログラムがプロセッサーとメモリを調べ、パフォーマンスレベルを決定します:
  - **低** — 遅いコンピューター用、限定的な AI 機能
  - **中** — 日常使用のバランスが取れた選択 (推奨)
  - **高** — 高速コンピューター用、すべての AI 機能
- プログラムの推奨をそのまま使用するのが通常は正しいです。**次へ**をクリックします。

### ステップ 3 — AI セットアップ (オプション)

ArchivistPro の **AI Chat** 機能には AI サポートが必要です。ここで 3 つの選択肢があります:

| 選択肢 | 利点 | 欠点 |
|---|---|---|
| **ローカル AI (Ollama)** | 完全オフライン、プライベート; データは外に出ない | Ollama を別途インストールする必要がある |
| **クラウド AI** | 高速、インストール不要 | インターネット必要、API キー必要 |
| **スキップ** | 最も簡単 | AI 機能は動作しません |

ヒント: **ローカル AI** を選択。ウィザードは [ollama.com](https://ollama.com/download) にあなたを送ります。Ollama をダウンロードしてインストールし、ArchivistPro に戻って**「再チェック」**をクリックします。

今 AI をスキップしたい場合は**スキップ**を選択。後で**設定 > AI** から有効にできます。

### ステップ 4 — DWG サポート (オプション)

アーカイブに DWG (AutoCAD 図面) ファイルがある場合、**ODA File Converter** という小さなヘルパーをインストールすることをお勧めします。これにより ArchivistPro が DWG ファイルの内容 (レイヤー、ブロック、テキスト) を理解できるようになります。

- ODA が検出された場合は自動 → **次へ**
- インストールされていない場合は「**ダウンロード & インストール**」をクリック、またはこのステップをスキップ

### ステップ 5 — 概要と準備完了

![ウィザード最終ステップ](img/install/wizard-step-5.png)

選択内容の概要が表示されます。すべて正しければ**開始**をクリックします。

## 5. 管理者アカウントの作成

![管理者セットアップ](img/install/admin-setup.png)

ウィザードの後、**管理者アカウント作成**画面が表示されます。**このアカウントはあなたが作成します** — プログラムにはあらかじめ設定されたユーザー名/パスワードはありません。

1. **ユーザー名** — ユーザー名を選択 (例: "ahmet" または "boss")。3〜32 文字。
2. **パスワード** — 強力なパスワードを選択。忘れないでください。
   - 6 文字以上 (12 文字以上推奨)
   - 文字と数字の組み合わせが良い
3. **パスワード確認** — 同じパスワードを再入力
4. **アカウント作成**をクリック

> **パスワードを忘れた場合:** プログラムは自動的に「リカバリーキー」ファイルを作成し、次の場所に保存します:
> `C:\Users\<あなたの名前>\AppData\Roaming\com.archivistpro.desktop\recovery.key`
>
> **このファイルを安全な場所にバックアップ**してください (USB にコピーするか、個人のメールに送信)。パスワードを忘れた場合、このファイルでリセットできます。

## 6. 最初のファイルを追加

プログラムを開くと空の画面が表示されます。アーカイブにファイルを追加するには:

![フォルダースキャンボタン](img/install/scan-folder-button.png)

1. 左パネルで**「フォルダーをスキャン & インデックス作成」**をクリック
2. 建築ファイルが含まれるフォルダーを選択 (例: `D:\Projects`)
3. **「スキャン開始」**をクリック
4. スキャンが自動的に開始 — プログレスバーが表示されます。所要時間はファイル数によって異なります (1000 ファイル ≈ 5 分)

![スキャン進行状況](img/install/scan-progress.png)

スキャンが完了すると、メインリストにファイルが表示されます。検索、タグ付け、ソートが可能になります。

## 7. よくある質問 (FAQ)

### 古いバージョンから v3.0.0 にアップグレードしました。ファイルはどうなりますか?

初回起動時、古いアーカイブが**自動的に新しい V3 アーキテクチャに移行**されます。数秒かかります。データは安全です — ロールバック用のバックアップファイル (`archivist_premigrate_v3.db.bak`) が自動的に保持されます。

### インターネットなしでプログラムを使えますか?

はい。インストール後、インターネットは**不要**です。ローカル AI (Ollama) を選択した場合、AI も完全にオフラインで動作します。クラウド AI を選択した場合、その機能のみインターネットが必要です。

### ファイルを別のコンピューターに移動できますか?

はい。プログラムの**設定 > ネットワーク > エクスポート / レポート**メニューから、アーカイブを `.archivistpro` 形式でエクスポートできます。このファイルを新しいコンピューターに持って行き、**「インポート (.archivistpro)」**でロードしてください。

### バックアップはどう機能しますか?

プログラムは各スキャン前に自動的にバックアップを取ります (最後の 5 つを保持)。**設定 > ストレージ**メニューから手動でバックアップを取得・復元できます。v3.0.0 以降、バックアップには `archivist.db` と `archivist_vec.db` の両方が**一緒に**含まれます。

### インストーラーが失敗しました。どうすればよいですか?

1. ダウンロードしたファイルを右クリック → **プロパティ** → 「**ブロックを解除**」チェックボックスが存在する場合はチェック → **OK**
2. インストーラーを再試行
3. それでも失敗する場合、ウイルス対策ソフトがブロックしている可能性があります。ArchivistPro を例外として追加してください
4. 問題が解決しない場合は、GitHub の Issues ページに投稿してください: https://github.com/ahmet3ddd/Arsiv-H2/issues

### プログラムをアンインストールしたいです。どうすれば?

Windows **設定 > アプリ > インストールされているアプリ**メニューで「**ArchivistPro**」を見つけて → **アンインストール**。データはこのフォルダーに残ります (完全に削除する場合は手動で削除):
`C:\Users\<あなたの名前>\AppData\Roaming\com.archivistpro.desktop\`
