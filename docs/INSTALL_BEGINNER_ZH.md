# ArchivistPro — 安装指南 (面向初学者)

> **版本:** 3.0.0 | **日期:** 2026-05-23 | **平台:** Windows 10/11 (64-bit)
>
> 如果您是第一次在计算机上安装程序或安装经验有限，本指南适合您。每个步骤都配有截图说明；没有专业术语，无需先备知识。
>
> 如需更简洁/技术性的摘要，请参阅 **[专业安装指南](https://github.com/ahmet3ddd/ArchivistPro/releases/download/v3.0.0/INSTALL_PRO_ZH.md)**。

## 1. 准备工作

### 1.1. 您的计算机准备好了吗？

在安装 ArchivistPro 之前，请检查您的计算机是否满足以下要求。大多数现代 PC 都能轻松满足。

| 要求 | 最低 | 推荐 |
|---|---|---|
| 操作系统 | Windows 10 (64-bit) | Windows 11 (64-bit) |
| RAM (内存) | 4 GB | 8 GB 或更多 |
| 可用磁盘空间 | 2 GB | 5 GB 或更多 |
| 处理器 (CPU) | x64 架构 (大多数 Intel / AMD) | 4 核或更多 |
| 显示器 | 1366×768 像素 | 1920×1080 或更高 |

> **如何查看我的计算机规格？**
>
> 1. 同时按 `Windows 键 + R`
> 2. 在小对话框中，键入 `msinfo32` 并单击 **确定**
> 3. 在"系统摘要"下，您将看到操作系统、处理器和内存信息

### 1.2. 管理员权限

要安装程序，您必须以**管理员**身份登录 Windows。如果您使用公司电脑并看到"需要管理员权限"，请联系 IT 部门。

### 1.3. 互联网连接

互联网**仅在下载安装程序时**需要。安装完成后，ArchivistPro **离线运行** — 您的文件不会发送到您的计算机外。

## 2. 下载安装程序

![GitHub Releases 页面](img/install/github-releases.png)

1. 打开您的 Web 浏览器 (Chrome、Edge、Firefox)
2. 转到地址: **https://github.com/ahmet3ddd/Arsiv-H2/releases/latest**
3. 在"**Assets**"部分下您将看到文件。下载**其中之一**:
   - **`ArchivistPro_3.0.0_x64_en-US.msi`** ← 推荐
   - `ArchivistPro_3.0.0_x64-setup.exe` ← 备选

> **MSI 与 EXE 的区别:** 两者安装相同的程序。MSI 更常见，在公司计算机上更易于管理。下载哪一个都可以。

> **如果浏览器警告您:** 可能会出现类似"很少下载此文件；您确定吗？"的消息。单击**保留**或**允许**继续下载。

## 3. 运行安装程序

![安装向导](img/install/installer-wizard.png)

1. **双击**您下载的文件 (`ArchivistPro_3.0.0_x64_en-US.msi`)。通常位于**下载**文件夹中
2. Windows 可能会询问: "*您是否要允许此应用对您的设备进行更改？*" → 选择**是**
3. 在安装程序窗口中:
   - 单击**下一步**
   - 阅读并接受许可证，然后**下一步**
   - 保留默认安装位置 (`C:\Program Files\ArchivistPro\`) 并单击**下一步**
   - 单击**安装**
4. 等待几秒钟。出现"安装完成"时，单击**完成**

> **如果您看到 SmartScreen 警告:** 可能会出现标题为"Windows 已保护您的电脑"的蓝色屏幕。这是因为该应用程序正在进行数字签名。请执行以下步骤:
>
> 1. 单击"**更多信息**"
> 2. 单击"**仍要运行**"

安装后，桌面上会出现 **ArchivistPro** 快捷方式。您也可以从"开始"菜单打开它。

## 4. 首次启动 — 设置向导

![设置向导步骤 1](img/install/wizard-step-1.png)

首次打开程序时，**5 步设置向导**会迎接您。它只显示一次。大约 3-5 分钟。

### 步骤 1 — 语言与系统检查

- 选择界面语言: **中文** (您可以随时从**设置**更改)
- 程序自动检查您的硬件。如果没有错误，请单击**下一步**

### 步骤 2 — 硬件检测

![硬件检测](img/install/wizard-step-2.png)

- 程序检查您的处理器和内存，并确定您的性能级别:
  - **低** — 用于较慢的计算机，AI 功能有限
  - **中** — 日常使用的良好平衡 (推荐)
  - **高** — 用于快速计算机，所有 AI 功能
- 保留程序的建议通常是正确的。单击**下一步**

### 步骤 3 — AI 设置 (可选)

对于 **AI 聊天**功能，ArchivistPro 需要 AI 支持。您有 3 个选项:

| 选项 | 优势 | 劣势 |
|---|---|---|
| **本地 AI (Ollama)** | 完全离线，私密；数据不外流 | 需要单独安装 Ollama |
| **云 AI** | 快速，无需安装 | 需要互联网，需要 API 密钥 |
| **跳过** | 最简单 | AI 功能无法工作 |

提示: 选择**本地 AI**。向导会将您发送到 [ollama.com](https://ollama.com/download)；下载并安装 Ollama，然后返回 ArchivistPro 并单击**"重新检查"**。

如果您现在想跳过 AI，请选择**跳过**。您可以稍后从**设置 > AI** 启用它。

### 步骤 4 — DWG 支持 (可选)

如果您的档案包含 DWG (AutoCAD 图纸) 文件，建议安装一个名为 **ODA File Converter** 的小型助手。这使 ArchivistPro 能够理解 DWG 文件的内容 (图层、块、文本)。

- 如果检测到 ODA，则自动 → **下一步**
- 如果未安装，请单击"**下载并安装**"，或跳过此步骤

### 步骤 5 — 摘要与就绪

![向导最后步骤](img/install/wizard-step-5.png)

显示您的选择摘要。如果一切正常，请单击**开始**

## 5. 创建管理员帐户

![管理员设置](img/install/admin-setup.png)

在向导之后，出现**管理员帐户创建**屏幕。**您创建此帐户** — 程序中没有预设的用户名/密码。

1. **用户名** — 选择一个用户名 (例如 "ahmet" 或 "boss")。3-32 个字符
2. **密码** — 选择一个强密码。不要忘记它
   - 至少 6 个字符 (推荐 12+)
   - 字母和数字混合好
3. **确认密码** — 再次键入相同的密码
4. 单击**创建帐户**

> **如果您忘记密码:** 程序会自动创建一个"恢复密钥"文件并将其保存到:
> `C:\Users\<您的名字>\AppData\Roaming\com.archivistpro.desktop\recovery.key`
>
> **请将此文件备份**到安全的地方 (复制到 USB 驱动器，或通过电子邮件发送到您的个人邮箱)。如果您忘记密码，可以使用此文件重置。

## 6. 添加您的第一批文件

打开程序时，您会看到一个空屏幕。要向您的档案添加文件:

![扫描文件夹按钮](img/install/scan-folder-button.png)

1. 在左侧面板中，单击**"扫描并索引文件夹"**
2. 选择包含您的建筑文件的文件夹 (例如 `D:\Projects`)
3. 单击**"开始扫描"**
4. 扫描自动开始 — 出现进度条。持续时间取决于文件数量 (1000 个文件 ≈ 5 分钟)

![扫描进度](img/install/scan-progress.png)

扫描完成后，您的文件会出现在主列表中。现在您可以搜索、标记和排序它们。

## 7. 常见问题 (FAQ)

### 我从旧版本升级到 v3.0.0，我的文件会怎样？

首次启动时，您的旧档案会**自动迁移到新的 V3 架构**。这需要几秒钟。您的数据是安全的 — 自动保留一个备份文件 (`archivist_premigrate_v3.db.bak`) 以供回滚。

### 我可以在没有互联网的情况下使用程序吗？

是的。安装后**不需要**互联网。如果您选择本地 AI (Ollama)，AI 也会完全离线工作。如果您选择云 AI，仅对该功能需要互联网。

### 我可以将文件移动到另一台计算机吗？

是的。从**设置 > 网络 > 导出 / 报告**菜单，您可以将您的档案导出为 `.archivistpro` 格式。将此文件带到新计算机并使用**"导入 (.archivistpro)"** 加载它。

### 备份如何工作？

程序在每次扫描前自动备份 (保留最近 5 个)。您可以从**设置 > 存储**菜单手动备份和恢复。从 v3.0.0 开始，备份**一起**包含 `archivist.db` 和 `archivist_vec.db` 文件。

### 安装失败了，我该怎么办？

1. 右键单击您下载的文件 → **属性** → 如果存在"**解除阻止**"复选框，请勾选 → **确定**
2. 再次尝试安装
3. 如果仍然失败，您的杀毒软件可能正在阻止；将 ArchivistPro 添加到例外列表
4. 如果问题仍然存在，请在 GitHub Issues 页面上提交问题: https://github.com/ahmet3ddd/Arsiv-H2/issues

### 我想卸载程序，怎么做？

打开 Windows **设置 > 应用 > 已安装的应用**菜单，找到"**ArchivistPro**" → **卸载**。您的数据保留在以下文件夹中 (如果要完全清除，请手动删除):
`C:\Users\<您的名字>\AppData\Roaming\com.archivistpro.desktop\`
