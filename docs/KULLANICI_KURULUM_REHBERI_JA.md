# ArchivistPro — エンドユーザー セットアップガイド

**バージョン：** 2.2.1 | **プラットフォーム：** Windows 10/11（64 ビット）

---

## 1. システム要件

| 要件 | 最小 | 推奨 |
|---|---|---|
| オペレーティングシステム | Windows 10（64 ビット） | Windows 11（64 ビット） |
| RAM | 4 GB | 8 GB 以上 |
| ディスク空き容量 | 2 GB | 5 GB 以上 |
| プロセッサー | x64 対応 CPU | 4 コア以上 |

> Node.js、Rust、その他の開発ツールは**不要**です — 標準的な Windows インストールで十分です。

---

## 2. インストーラーのダウンロード

- GitHub Releases ページにアクセス：`https://github.com/ahmet3ddd/Arsiv-H2/releases/latest`
- **`ArchivistPro_*_x64_en-US.msi`**（MSI、推奨）または **`ArchivistPro_*_x64-setup.exe`**（EXE）をダウンロードしてください。

> `.sig` 拡張子のファイルをダウンロードする必要はありません。これらは自動更新の検証にのみ使用されます。

---

## 3. インストーラーの実行

1. ダウンロードした `.msi` または `.exe` ファイルをダブルクリックします。
2. Windows が「このアプリがデバイスに変更を加えることを許可しますか？」と尋ねたら **はい** をクリックします。
3. ウィザードの指示に従います：**次へ → インストール → 完了**。
4. インストール完了後、デスクトップのショートカットまたはスタートメニューから ArchivistPro を起動できます。

> **SmartScreen の警告が表示された場合：** 「詳細情報」→「実行する」をクリックしてください。この警告はコード署名証明書がまだ有効でないために表示されますが、アプリケーションは安全です。

---

## 4. 初回起動 — セットアップウィザード（5 ステップ）

アプリケーションを初めて起動すると、**1 回限りの**セットアップウィザードが実行されます（約 5 分）。

### ステップ 1 — 言語とシステムチェック
- インターフェース言語を選択します：**トルコ語** または **英語**。
- アプリがハードウェアと Windows バージョンを自動的に確認します。

### ステップ 2 — ハードウェア検出
- CPU、RAM、パフォーマンスを計測し、AI に適切なハードウェアレベルを決定します。
- 検出されたレベル（低 / 中 / 高）は自分で変更することもできます。

### ステップ 3 — AI セットアップ *（任意）*
- **Ollama** がコンピューターで実行中の場合、自動的に検出されてローカル AI が有効になります。
- Ollama がインストールされていない場合はこのステップをスキップできます。AI 機能なしでもアプリは完全に動作します。

### ステップ 4 — DWG サポート *（任意）*
- **ODA FileConverter** がインストールされている場合、自動的に検出されて高度な DWG プレビューが有効になります。
- インストールされていない場合は、ウィザードからワンクリックでインストールするか、このステップをスキップできます。

### ステップ 5 — サマリーと準備完了
- 選択した設定のサマリーが表示されます。すべて正しければ **開始** ボタンをクリックします。

---

## 5. 初回ログイン — 管理者アカウントの作成

ウィザード完了後、ユーザーデータベースが空のため、**最初の管理者アカウント**を作成する画面が表示されます。

1. **ユーザー名**を入力します。
2. **パスワード**を設定します（最低 6 文字）。
3. パスワードを確認して **アカウントを作成** ボタンをクリックします。
4. 作成した認証情報でログインします。

> パスワードを忘れた場合、回復キーが次の場所に自動保存されています：
> `C:\Users\<ユーザー名>\AppData\Roaming\com.archivistpro.desktop\recovery.key`

---

## 6. 初回使用 — フォルダーのスキャン

ログイン後、アプリをすぐに使い始められます：

1. 左パネルの **スキャン** ボタンをクリックします。
2. アーカイブしたいフォルダーを選択します（DWG、RVT、MAX、IFC、PDF などを含むフォルダー）。
3. スキャン完了後、ファイルが自動的にインデックスされ、プレビューが生成されます。

---

## 7. 任意：Ollama のインストール（AI 機能）

AI 搭載の検索と OCR 機能を使いたい場合：

1. `https://ollama.com` から Ollama をダウンロードしてインストールします。
2. コマンドプロンプトを開き、ビジョンモデルを取得します：
   ```
   ollama pull llava
   ```
3. Ollama をバックグラウンドで実行したまま ArchivistPro を再起動すると、AI 機能が自動的に有効になります。

> Ollama なしでもアプリは**完全に機能します**。無効になるのは LLM ベースの検索と OCR のみです。

> **注意：** 視覚的類似検索（CLIP）は追加設定が不要です。初回スキャン時に AI モデル（約 87 MB）が自動ダウンロードされ、以降はキャッシュから読み込まれます。この機能は Ollama とは独立して動作します。

---

## 8. 任意：ODA FileConverter のインストール（高度な DWG 対応）

DWG ファイルからより高品質なプレビューとメタデータを取得したい場合：

1. `https://dl.opendesign.com` から ODA FileConverter をダウンロードします。
2. ダウンロードしたインストーラーを実行して完了させます。
3. ArchivistPro を再起動すると DWG サポートが自動検出されます。

> このステップは完全に任意です。ArchivistPro には組み込みの DWG リーダーがあるため、インストールしなくても動作します。

---

## 9. 自動更新

- 新しいバージョンがリリースされると、アプリが通知で知らせます。
- **設定 → アップデート** から手動でも最新バージョンを確認できます。
- 更新ファイルのダウンロード後は、アプリを再起動するだけです。

---

## 10. トラブルシューティング

| 問題 | 解決策 |
|---|---|
| アプリが起動しない | ウイルス対策ソフトが ArchivistPro をブロックしていないか確認し、必要に応じて例外を追加してください。 |
| SmartScreen の警告 | 「詳細情報」→「実行する」を使用してください。 |
| DWG プレビューが表示されない | ODA FileConverter のインストールを確認してください（ステップ 8）。 |
| AI 機能が動作しない | Ollama がバックグラウンドで実行されているか確認してください（`ollama serve`）。 |
| パスワードを忘れた | ログイン画面で `%APPDATA%\com.archivistpro.desktop\recovery.key` ファイルを使用してください。 |
| スキャンが非常に遅い | スキャン中は他の重いプログラムを閉じてください。RAM を増やすとパフォーマンスが向上します。 |
